    📝 Informations sur l’étude comparative#

    On récupère l’impact environnementale d’un aliment sur la base de données Agribalyse 3.2, puis on récupère le nombre de protéine présent dans cet aliment sur la base de données Ciqual de l’Anses et on en déduit l’impact environnemental pour produire 1g de protéine de cet aliment. Cette étude montre que produire 10g de protéine de boeuf est presque 20x plus impactant que produire 10g de protéine de lentilles vertes. A pollution égale on peut donc obtenir : - 10g de protéine de boeuf - 70g de protéine d’oeuf - 195g de protéine de lentilles vertes
